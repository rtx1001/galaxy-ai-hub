# Galaxy AI Hub Characters

Each character gets a folder here.

- `soul.md` stores the character's persistent identity, emotional cognition, speech style, boundaries, and extra notes.
- `settings.json` stores app-level character settings such as voice sample path, avatar metadata, and the editable prompt snapshot.

The app creates these files automatically when a character is selected. You can edit `soul.md` directly to make a character feel less generic and more like a continuous companion.
